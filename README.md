# Lazada Price Tracker 🛒

สคริปต์ Python สำหรับดึงราคาสินค้าจาก Lazada อัตโนมัติ
และบันทึกลงไฟล์ Excel พร้อมวันที่และเวลา

## ทำอะไรได้บ้าง
- ดึงชื่อสินค้าและราคาจาก Lazada อัตโนมัติ
- บันทึกลง Excel พร้อมวันที่และเวลาที่ดึงข้อมูล
- รองรับการค้นหาสินค้าได้ทุกประเภท

## วิธีใช้งาน
1. ติดตั้ง Library ที่จำเป็น
2. แก้ชื่อสินค้าที่ต้องการค้นหาในโค้ด
3. รันคำสั่ง python price.py
4. เปิดไฟล์ lazada_price.xlsx ดูผลลัพธ์ได้เลย

## ติดตั้ง Library
pip install selenium webdriver-manager pandas

## ผลลัพธ์ที่ได้
| คอลัมน์ | รายละเอียด |
|---|---|
| ชื่อสินค้า | ชื่อสินค้าจาก Lazada |
| ราคา | ราคาสินค้าปัจจุบัน |
| วันที่ | วันที่และเวลาที่ดึงข้อมูล |
